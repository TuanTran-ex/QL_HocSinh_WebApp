# QL_HocSinh_WebApp
Web Quản lý thông tin học sinh và lớp học  
Được xây dựng dựa trên ExpressJS, MongoDB và NodeJS

## Hướng dẫn chạy Project  
### Cài đặt node modules
```shell
npm install
```
### Khởi động Web
```shell
npm start
```
